# 什么是Twig.js?


Twig.js是[Twig](https://twig.symfony.com/)的纯JavaScript实现PHP模板语言

目标是提供一个与浏览器和服务器端容器（如node.js）兼容的库。Twig.js目前正在开发中，支持Twig模板语言的一个有限子集（未来还会有更多）。

## 新闻

- 6/10/2016 - Twig 现在可以作为 [Keystone.js](http://keystonejs.com/) 中的模板引擎使用
